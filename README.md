# PySpeechColab

A library of speech gadgets.

## Installation

Fork/clone the GitHub repo:

    git clone git@github.com:SpeechColab/PySpeechColab.git
    cd PySpeechColab

Building a virtual environment is highly recommended:

    python3 -m venv venv
    . venv/bin/activate

Then install with pip:

    pip install -e '.[dev]'

Run unit tests:

    pytest test
